# cat-dog-svm-classifier

Cat vs. dog image classification using a Support Vector Machine (SVM). A classic
computer-vision baseline: extract image features, train an SVM, and evaluate.

## Contents

- `SVM_cat_and_dog.ipynb` — feature extraction, SVM training, and evaluation.

## Run

```bash
jupyter notebook SVM_cat_and_dog.ipynb
```

## Tech

Python, scikit-learn (SVM), image classification.
